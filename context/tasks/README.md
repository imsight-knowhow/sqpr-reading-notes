# Tasks and Work Items

## HEADER
- **Purpose**: Track project tasks organized by status and scope
- **Status**: Active
- **Date**: 2025-12-09
- **Dependencies**: Plans in `context/plans`, design docs, and open issues
- **Target**: AI assistants, developers, and maintainers

## Content
Use this directory to manage work items as markdown files. Place tasks that are currently being worked on in `working/`, completed tasks in `done/`, and planned or unstarted tasks in `backlog/`. Keep task documents small, focused, and linked to related plans or issues.
